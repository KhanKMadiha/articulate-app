# Articulate

**[Try it live →](https://articulate-app-production.up.railway.app)**

A daily reading practice app for professionals who want to speak with more confidence. Articulate generates a personalised passage every day using AI, tailored to your career goals and the way you want to sound — then lets you record yourself and get scored on accuracy, fluency and pace.

Built as a full-stack PWA using React, Express and the Anthropic Claude API. Started in Cursor, continued with Claude Code.

---

## The idea

A coffee chat gave me this idea.

I was speaking to a senior engineering manager at a major tech company and he said something that stuck with me:

> "You need to learn how to speak different languages. To customers, to engineers, to execs. They all need something different."

Most of us know what we want to say. The gap is how we say it — the confidence, the clarity, the delivery. Articulate is my attempt to make that easier to practise, every day.

---

## What it does

- **Personalised daily passages** — generated by Claude based on your job title, industry, career goals and focus areas
- **Speaking inspirations** — choose speakers you admire and passages are written in their spirit
- **Record and score** — read aloud, record yourself, and get AI feedback on accuracy, fluency and pace
- **Daily habit** — streak tracking, a motivational moment after each session, and optional push notification reminders
- **Two flows** — get straight to your passage, or do a quick energy and theme check-in first
- **Library** — save favourite passages for a second read or rehearsal before a big moment

## Running locally

### Prerequisites
- Node.js 20+
- An Anthropic API key from [console.anthropic.com](https://console.anthropic.com)

### Setup

1. Clone the repo
   ```bash
   git clone https://github.com/KhanKMadiha/articulate-app.git
   cd articulate-app/Madiha/Articulate
   ```

2. Create a `.env` file in the root
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```

3. Install dependencies and start
   ```bash
   npm run dev
   ```

This starts the Express API on port 3001 and the Vite dev server on port 5173. Open `http://localhost:5173`.

## Deployment

The app is configured to deploy on Railway using Docker.

1. Push to the `main` branch of your connected GitHub repo
2. In Railway, set **Root Directory** to `Madiha/Articulate`
3. Set **Builder** to Dockerfile
4. Add environment variables in Railway:
   - `ANTHROPIC_API_KEY`
   - `VAPID_PUBLIC_KEY` *(optional — for push notifications)*
   - `VAPID_PRIVATE_KEY` *(optional — for push notifications)*
   - `VAPID_CONTACT_EMAIL` *(optional — your email, used in push notification headers)*

For push notifications, generate VAPID keys with:
```bash
npx web-push generate-vapid-keys
```
